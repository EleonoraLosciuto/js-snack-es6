# Array & Objects Snacks (ES6)

This is the 18th exercise I have completed as part of the web development master. It focuses on working with arrays of objects, dynamic property assignment, and leveraging modern ES6 syntax in JavaScript.
Repository name: `js-snack-es6`

## 📝 Task

### Snack 1: Racing Bikes
* Create an array of objects, where each object represents a racing bike with the following properties: `name` and `weight`.
* Find and print to the screen the bike with the lowest weight.

### Snack 2: Football Teams
* Create an array of objects representing football teams. Each team object must contain the properties: `name`, `points`, and `fouls suffered`.
* Initialize the `name` property with real data, while setting `points` and `fouls suffered` to `0`.
* Generate random numbers to replace the initial `0` values for both `points` and `fouls suffered`.
* Create a new filtered array containing only the `name` and `points` properties for each team, then print the final result to the browser console.
### Bonus
* I decided to challenge myself by sorting the teams and create a ranking (table), showing team, points and position
